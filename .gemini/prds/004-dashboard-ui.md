# PRD: 통합 대시보드 및 활동 로그 (Dashboard)

## Metadata
- **Status**: Draft
- **Depends On**: PRD-002, PRD-003
- **Unblocks**: PRD-005

## 1. 개요
분석된 러닝 데이터를 시각화하고 탐색하는 메인 대시보드입니다.

## 2. 화면 구성

### 2.1 KPI 섹션 (Top)
- **카드 1: VDOT & 예측**: 현재 VDOT 점수와 마라톤 예상 시간 표시.
- **카드 2: 훈련 부하**: 이번 주 거리(km)와 ACWR 상태(안전/위험).
- **카드 3: 꾸준함**: 최근 4주간의 훈련 빈도 점수.

### 2.2 차트 섹션 (Middle)
- **주간 훈련 & 부하 (Mixed Chart)**:
    - X축: 주(Week).
    - Y1축(Bar): 주간 총 거리.
    - Y2축(Line): ACWR 수치.
    - 의미: "내가 얼마나 뛰었고, 그게 몸에 얼마나 무리가 갔는지" 비교.

### 2.3 데이터 그리드 (Bottom)
- **활동 로그**: 전체 활동 내역을 리스트로 표시.
- **기능**: 검색, 정렬, 필터링(Run만 보기 등).

## 3. UI/UX 요구사항
- **반응형**: 모바일에서는 그리드를 숨기거나 간소화된 리스트 뷰로 대체.
- **인터랙션**: 차트의 바를 클릭하면 해당 주의 활동 로그만 필터링됨.
