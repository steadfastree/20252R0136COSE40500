# PRD: 스마트 훈련 플래너 (Training Planner)

## Metadata
- **Status**: Draft
- **Depends On**: PRD-002 (DB)
- **Unblocks**: PRD-006

## 1. 개요
사용자가 훈련 일정을 수립하고 관리하는 캘린더 도구입니다.

## 2. 사용자 시나리오
1.  **계획 수립**: 캘린더의 빈 날짜를 클릭하여 "10km 조깅" 계획을 추가한다.
2.  **가이드 받기**: 계획 입력 시, "현재 VDOT 기준 5:30/km 페이스가 적절합니다"라는 추천을 받는다.
3.  **확인**: 이번 달에 계획된 훈련이 얼마나 남았는지 한눈에 본다.

## 3. 기능 요구사항

### 3.1 캘린더 인터페이스
-   월간 뷰(Month View) 기본 제공.
-   일정 드래그 앤 드롭으로 날짜 변경 가능.

### 3.2 훈련 CRUD
-   **생성**: 날짜, 타입(Run, Interval 등), 목표(거리/시간), 메모 입력.
-   **상태 관리**: `Scheduled` (예정됨) 상태로 시작.

### 3.3 스마트 가이드 (VDOT Integration)
-   입력 모달에 "추천 페이스 불러오기" 버튼 제공.
-   클릭 시 최신 VDOT 데이터를 기반으로 페이스 자동 입력.

## 4. 성공 기준
-   [ ] 계획 생성 시 DB에 정상적으로 저장됨.
-   [ ] 캘린더에서 날짜 이동 시 즉각적으로 반영됨 (Optimistic Update).
