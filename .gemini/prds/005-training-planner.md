# PRD: 스마트 훈련 플래너 (Training Planner)

## 1. 개요
사용자가 자신의 목표(예: 마라톤 서브-4)를 달성하기 위해 훈련 일정을 수립하고 관리하는 도구입니다. 막연한 다짐이 아닌 구체적인 날짜와 목표가 있는 '계획'으로 전환을 돕습니다.

## 2. 주요 기능

### 2.1 캘린더 뷰 (Calendar View)
- 월간/주간 뷰를 제공하여 훈련 흐름을 파악.
- 날짜별 셀(Cell)에 **계획된 훈련(Plan)**과 **실제 수행한 훈련(Actual)**을 동시에 표시.
- 드래그 앤 드롭으로 훈련 일정을 손쉽게 변경.

### 2.2 훈련 생성 및 편집 (CRUD)
- **훈련 타입 선택**: 조깅, 인터벌, 템포런, 장거리주(LSD), 휴식 등.
- **목표 설정**: 거리(km) 또는 시간(분), 목표 페이스 설정.
- **반복 일정**: "매주 화요일 인터벌" 같은 반복 훈련 설정 기능 (V2 고려).

### 2.3 VDOT 기반 가이드 (Smart Guide)
- 훈련 계획 생성 시, PRD-003의 VDOT 엔진을 활용하여 적정 페이스를 제안해줌.
- 예: 사용자가 "인터벌"을 선택하면 "현재 VDOT 기준 4:15/km 페이스를 추천합니다"라고 표시.

## 3. 데이터 모델링 (Supabase)
- **`plans` 테이블**:
    - `id`, `user_id`, `date`, `type`, `target_distance`, `target_time`, `target_pace`, `notes`.
    - `is_completed` (수동 체크 또는 자동 매칭 여부).

## 4. UI/UX
- **직관적인 입력 폼**: 복잡한 설정 없이 몇 번의 클릭으로 훈련 등록.
- **시각적 구분**: 훈련 강도나 타입에 따라 캘린더 이벤트의 색상을 다르게 표시.

## 5. 성공 기준
- 캘린더에서 훈련 계획을 생성, 수정, 삭제하는 인터랙션이 매끄러워야 함.
- 모바일에서도 캘린더 일정 확인 및 간단한 추가가 가능해야 함.
